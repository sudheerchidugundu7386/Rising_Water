Flood Prediction

About the Project

The Flood Prediction System uses Machine Learning to analyze rainfall, temperature, humidity, cloud cover, and seasonal weather conditions to estimate flood risk. It provides fast and reliable predictions through a simple and user-friendly web application.

🌦 Climate Insights

Monitors important environmental factors including temperature, humidity, cloud cover, and seasonal rainfall to understand weather conditions associated with flood events.

🧠 Smart Prediction

A trained Random Forest Machine Learning model analyzes historical weather data and predicts the likelihood of flood occurrence with high accuracy.

⚡ Instant Results

Simply enter the required weather information and receive an immediate prediction through an easy-to-use web application.

Developed using Python | Flask | Machine Learning | Random Forest
